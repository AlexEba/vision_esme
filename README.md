Installation:
----------------

    git clone https://github.com/laurent-george/vision_esme.git
    cd vision_esme
    ./run.sh


Ouvrez dans votre navigateur web la page: [http://localhost:8888/tree?](http://localhost:8888/tree?) en utilisant le token affiché dans le terminal.
